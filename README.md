1. Unzip the folder
2. Run 'mix escript.build'
3. Run->  "./proj2 numberOfNodes topology algorithm"

4. Bonus -> "./proj2 numberOfNodes topology algorithm percentage"  Example -> "./proj2 100 honeycomb gossip 20"
